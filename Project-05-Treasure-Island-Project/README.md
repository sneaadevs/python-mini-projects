# 🏝️ Treasure Island

A simple text-based adventure game built with Python. Make the right choices to navigate through the island and find the hidden treasure. One wrong move, and it's game over!

## Features
- Interactive command-line gameplay
- Multiple decision paths
- Win and lose scenarios
- Beginner-friendly Python project

## How to Run
1. Make sure Python 3 is installed.
2. Clone the repository or download the project.
3. Run:

```bash
python treasure_island.py
```
