# 🎢 Rollercoaster Ticket Calculator

A simple Python program that calculates the price of a rollercoaster ticket based on the user's height, age, and photo preference.

### Concepts Used

* Variables
* User Input
* Conditional Statements (`if`, `elif`, `else`)
* Nested Conditions
* f-Strings

Built as part of the **100 Days of Code: Python Bootcamp** by Dr. Angela Yu.
