# Rock Paper Scissors 🎮

A simple command-line Rock Paper Scissors game built in Python as part of Angela Yu's **100 Days of Code: The Complete Python Pro Bootcamp**.

## Features
- Player chooses Rock, Paper, or Scissors.
- Computer makes a random choice.
- Displays ASCII art for each move.
- Determines whether the player wins, loses, or draws.

## Technologies Used
- Python
- `random` module

## How to Run
1. Clone this repository.
2. Run the Python file:
   ```bash
   python main.py
   ```
3. Enter:
   - `0` for Rock
   - `1` for Paper
   - `2` for Scissors

Enjoy playing! 🚀
